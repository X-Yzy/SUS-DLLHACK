## 6. 说明

最终代理源码会直接写入用户加载的 A() 模板。模板中已经包含 `#include <stdint.h>` 与 `#include <windows.h>` 时，程序不会再额外重复写入这两个头文件。

生成 `code.data` 时可以勾选 XOR 加密，默认密钥为 `0x5A`，也可以在界面中填写 `0-255` 或 `0x00-0xFF` 格式的自定义密钥。选择“默认模板：XOR 解密 code.data”后，模板会自动切换为 XOR 解密版 A()。

## 推荐：一键自动编译

以后优先运行项目根目录的 `build_all.bat`：

```powershell
C:\Users\Yzy12\Desktop\dllhack\build_all.bat
```

这个脚本会自动查找 `MinGW64`，配置 `DLLHACK_MINGW_ROOT`，编译主程序，部署 Qt 运行库，并检查 `build\tools\mingw64\bin\gcc.exe` 是否已经打包成功。这样每次重新编译后都能自动修复验证阶段找不到代理编译器的问题。
